# Dardo Pacífico

Website institucional da **Dardo Pacífico**, publicado através do GitHub Pages.

A empresa presta serviços de consultoria estratégica e desenvolvimento de projetos em áreas complexas, reguladas e de elevada exigência.

## Website

- Domínio principal: `https://dardopacifico.com`
- Repositório: `alexoxy/dardopacifico.github.io`
- Publicação: GitHub Pages, branch `main`, pasta `/ (root)`

## Estrutura

- `index.html` — página principal e formulário de contacto
- `obrigado.html` — página apresentada após o envio da mensagem
- `CNAME` — associação ao domínio `dardopacifico.com`

## Contacto

As mensagens enviadas através do formulário são encaminhadas para:

`info@dardopacifico.com`

O formulário utiliza o FormSubmit. No primeiro envio, será necessário confirmar o endereço através do email de ativação recebido nessa caixa.

## DNS

Para publicar o website com domínio próprio, devem ser alterados apenas os registos web necessários para o GitHub Pages. Não devem ser removidos ou alterados os registos MX, SPF, DKIM ou outros registos usados pelo Google Workspace.
